整体布局调整:
  在首页做一个 “三栏式布局”（类似 Notion 或现代邮件客户端）：左侧：极简的导航图标。中间：投资随笔的列表，卡片式设计，带有缩小的走势图。右侧：当前选中的投资思考详情，配合大面积留白和精美的排版。

一、审美风格
基于 Tailwind CSS v4 的极简工业风。
配色方案 (Palette)：
  背景：bg-zinc-50 (纸张感，减少屏幕疲劳)。
  文字：text-zinc-900 (主标题), text-zinc-500 (次要辅助信息)。
  强调色：使用 indigo-600 代替高饱和度红绿，仅在最终损益处保留柔和的 emerald-500 (跌) 和 rose-500 (涨)。
字体 (Typography)：
  数字：全站启用 font-mono 或 tabular-nums，确保金额、市盈率等数据垂直对齐，消除视觉抖动。
  正文：加大 line-height (leading-relaxed)，提升长篇随笔的阅读体验。

二、核心页面改造
1. 首页：从“资产大屏”到“逻辑看板” (The Thinking Hub)
修改建议：
  弱化资产数字：将“总资产/今日盈亏”移至右上角或左侧边栏底部，缩小字号。
  首屏核心：【未结案假设 (Open Hypotheses)】
  采用 Masonry (瀑布流) 或 卡片流。
  每张卡片展示：投资标的 + 核心逻辑摘录 + 记录时长 (如：已持仓 42 天) + 逻辑偏离度 (基于当前价格与目标价)。
  视觉补丁：在卡片背景加入极淡的 Recharts 迷你走势图 (Sparkline)，不显示坐标轴，只看趋势。
2. 写作页：从“记事本”到“实验存证” (The Lab Notebook)
修改建议：
  引入“快照”概念：在用户输入文字时，右侧自动浮动一个“实时数据面板”。一键存证按钮：点击后，将当前的股价、PE、PB、成交量等数据“冻结”在当前随笔中。
  逻辑标签体系：强制或引导选择维度：#宏观驱动、#基本面反转、#情绪博弈、#套利。
  沉浸模式：当用户开始输入长文时，自动隐藏侧边栏，只留白色的“纸张”区域。
3. 验证页：从“历史列表”到“跨时空对话” (Post-Mortem Review)
修改建议：
  双栏对比视图：左栏 (当初的我)：显示当时的随笔内容、存证时的股价、当时的市场情绪图标。
  右栏 (现在的市场)：显示当前股价、最新财报数据、实际走势。
  逻辑对错标记：设计三个“非金钱”维度的开关：逻辑证实、逻辑证伪、意外因素干扰。
  复盘金句：在底部增加一个高亮区域：“如果回到过去，我会改进什么？”
4.数据可视化的“呼吸感”
修改建议：
  曲线平滑：将折线图的 type 设置为 monotone，增加视觉上的流畅性。
  隐藏坐标轴：在小型的预览图（Sparklines）里，隐藏 X 轴和 Y 轴，只保留趋势线，能让页面显得更干净、高端。
  渐变填充：给折线下方加一层微弱的面积渐变（Area Gradient），会让图表更有层级感，不那么单薄。

下一步更新：
1.Tauri 桌面端增强：
  毛玻璃效果 (Backdrop Blur)：在侧边栏（Sidebar）或导航栏使用 Tailwind 的 backdrop-blur。这种“半透明”的质感会很有 macOS 原生应用的高级感。
  置顶小窗 (Global Shortcut)：利用 Tauri 的多窗口功能，按下 Alt + Space 弹出极简输入框，快速记录突发灵感，无需打开主程序。
  系统托盘：在 Tray Menu 显示最近一条逻辑验证的进度，时刻提醒自己“保持客观”。
2.逻辑漏斗：统计你哪类逻辑（如：基本面）的胜率最高，哪类（如：听消息）最常证伪。
3。PDF 研报导出：一键将某次完整的“思考-验证”过程导出为精美的 PDF，作为自己的投资编年史。
4.数据脱敏分享：只分享逻辑和比例，不分享具体金额，生成一张极具美感的分享卡片。
